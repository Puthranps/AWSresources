update-channel¶

Description¶

Updates a channel.

See also: AWS API Documentation

See 'aws help' for descriptions of global parameters.

Synopsis¶

  update-channel
--channel-id <value>
[--destinations <value>]
[--encoder-settings <value>]
[--input-attachments <value>]
[--input-specification <value>]
[--log-level <value>]
[--name <value>]
[--role-arn <value>]
[--cli-input-json <value>]
[--generate-cli-skeleton <value>]

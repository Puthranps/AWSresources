***
ses
***



===========
Description
===========

 

This document contains reference information for the `Amazon Simple Email Service <https://aws.amazon.com/ses/>`__ (Amazon SES) API, version 2010-12-01. This document is best used in conjunction with the `Amazon SES Developer Guide <http://docs.aws.amazon.com/ses/latest/DeveloperGuide/Welcome.html>`__ . 

 

.. note::

   

  For a list of Amazon SES endpoints to use in service requests, see `Regions and Amazon SES <http://docs.aws.amazon.com/ses/latest/DeveloperGuide/regions.html>`__ in the `Amazon SES Developer Guide <http://docs.aws.amazon.com/ses/latest/DeveloperGuide/Welcome.html>`__ .

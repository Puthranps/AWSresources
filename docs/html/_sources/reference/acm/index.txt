***
acm
***



===========
Description
===========

 

Welcome to the AWS Certificate Manager (ACM) API documentation.

 

You can use ACM to manage SSL/TLS certificates for your AWS-based websites and applications. For general information about using ACM, see the ` *AWS Certificate Manager User Guide* http://docs.aws.amazon.com/acm/latest/userguide/`__ .
